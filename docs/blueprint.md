# **App Name**: Gestión de Proveedores

## Core Features:

- Registro de Proveedores: Permite a los proveedores registrarse en la aplicación, incluyendo la recopilación de información como tipo de documento, número de documento, razón social/nombre, ciudad, departamento, país, dirección, correo electrónico y contraseña.
- Inicio de Sesión: Permite a los proveedores iniciar sesión en la aplicación utilizando su correo electrónico y contraseña.
- Recuperación de Contraseña: Proceso de recuperación de contraseña en caso de olvido, garantizando el acceso a la cuenta.
- Cuentas de Administrador: Agregar por defecto las cuentas de correo: sistemas@frioalimentaria.com.co y asistente@frioalimentaria.com como usuarios administradores de la aplicación.

## Style Guidelines:

- Primary color: Light desaturated blue (#165793) for a sense of trust and stability.
- Background color: Very light desaturated blue (#F0F4F7).
- Accent color: Muted teal (#0096D6) to highlight key actions and information.
- Font pairing: 'Inter' (sans-serif) for both headings and body text, providing a clean and modern look.
- Use simple, professional icons related to user accounts and security (e.g., key for password, profile for user info).
- Clean and intuitive layout with a focus on ease of use. Utilize clear labels and sufficient spacing.
- Subtle animations to provide feedback on user interactions (e.g., form submission, password recovery).
